# Dashboard Screenshots

## Traffic Table
Sortable interface table showing peak traffic, P95 max and link speed across all monitored devices. User can filter results using interface descriptions, hostnames and interface types. Click any row to drill into the single interface graph.

![Traffic Table](Traffic%20Table.png)

---

## Traffic Graphs
Per-interface time-series showing Traffic In / Out with 95th percentile reference line and total bytes transferred. User can filter results using interface descriptions, hostnames and interface types. 

![Traffic Graphs](Traffic%20Graphs.png)

---

## RTT Table
IP SLA probe latency summary — P25 through P95 percentiles per site. The 95th column is colour-coded: green below 6 ms, yellow to 20 ms, red above.

![RTT Table](RTT%20Table.png)

---

## Power Table
Per-device router power consumption showing current power value, 95th percentile and peak in Watts.

![Power Table](Power%20Table.png)

---

## Power Graph
Router power draw over time with P95 reference line. Loaded on row click from the Power Table.

![Power Graph](Power%20Graph.png)
